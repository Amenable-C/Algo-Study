# Week7 BOJ_2665_S4_미로만들기

## 접근방법
벽부수고 이동하기랑 똑같이 품. 단지 결과로 무엇을 출력하느냐가 다름.
(벽부수고는 그래서 큐사이즈만큼 돌렸는데 이건 그냥 돌림)
dfs탐색. 큐.
클래스를 만든다.(x,y,몇번 바꿨는지.)
도착했을때 바꾼횟수를 저장.
현재 길을 가는중인데 바꾼횟수가 이미 크면 가지치기.

<br>

## 실행 흐름
1. 입력받으면서 map 그리기
2. visited(해당 위치에 도착할 수 있는 최소 변경횟수) 초기화, (0,0) 큐에 넣기.
3. 큐에서 꺼내서 출발.
	1. 상하좌우 탐색
	2. 전환 횟수를 더 작게 갈수 있으면 이동.
4. 만약 도착지 도착하면 바꾼 횟수 update
