## Week10 BOJ 2573 G4 빙산

### 접근방법
1. 2중포문으로 빙산이 녹는 크기 구하기
2. 빙산 녹이기 ( 크기구하면서 녹이면 다른 곳에 영향이 가니까 크기 다 구하고 녹이기)
3. 몇 덩이로 나눠지는지 세기 ( bfs )
4. 2덩이 이상으로 나눠지면 끝내고 시간 출력
5. 0덩이면 다 녹았다는 거니까 끝내고 0출력


<br>

### 신경쓴점
