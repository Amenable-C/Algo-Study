## Week13 BOJ 2879 G3 코딩은 예쁘게

### 접근방법

- 숫자가 더 커지는 경우에만 차이만큼 더해준다.
- 부호가 바뀌는 경우에는 절댓값만큼 더한다.

다음과 같이 4가지 경우로 나누었다.

1. 직전 양수 또는 0, 현재 음수 : 
  현재 값의 절댓값을 result에 더한다.
2. 직전 음수 또는 0, 현재 양수 : 
  현재 값을 result에 더한다.
3. 직전 양수, 현재 양수 : 
  현재 값이 더 크다면 (현재 - 직전) 값을 result에 더한다.
4. 직전 음수, 현재 음수 : 
  절댓값으로 현재 값이 더 크다면 (직전 - 현재)값을 result에 더한다.


최종적으로 구해진 result를 출력한다.
