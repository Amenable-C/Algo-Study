## Week20 BOJ 12904 G5 A와B

### 접근방법

- S에서T를 만드는데는 2가지 방법이 있다.
    - 문자열의 뒤에 A를 추가한다.
    - 문자열을 뒤집고 뒤에 B를 추가한다.
- T에서 S를 만든다고 바꿔서 생각!
    -> 뒤에있는 A또는B를 없앰. B면 뒤집기. -> 1가지 방법!!
    -> 이것을 B길이랑 같아질 때 까지 하고, B와 같은지 비교
