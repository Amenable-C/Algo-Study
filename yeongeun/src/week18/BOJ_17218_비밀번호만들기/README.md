## Week18 BOJ 17196 G5 비밀번호 만들기

### 접근방법 포인트

- LCS(Longest Common Subsequence) 알고리즘 찾아봤다.
- dp배열을 만들어 최장 공통 부분 길이를 찾고,
- 해당 배열을 이용하여 문자열을 찾는다.
    
    뒤에서부터 시작해서 길이가 줄어드는 부분이 문자열에 포함되는 부분이 되어 추가해준다.

    문자열을 뒤에서부터 찾았으므로 다시 뒤집어주는 과정이 필요하다.
