## 풀이
1. 방별로 인접 리스트를 먼저 만들어줬다.
2. 이후 dfs탐색을 진행했는데, 노드 번호와 돈을 매개변수로 줬다.
3. 조건에 따라 진행하다가 노드 번호가 N이 되면 yes를 출력하고, 그전에 종료되면 No를 출력한다.
