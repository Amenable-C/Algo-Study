### 풀이
1. map을 bfs로 탐색하면서 열쇠, 문, 문서를 찾는다.
2. 문을 찾으면 해당하는 열쇠가 있는지 확인하고, 있으면 빈공간으로 변경한다.
3. 이때 열쇠가 없다면 따로 문을 저장 해놓는다. (나중에 열쇠를 발견하면 바로 열기 위함)
4. 열쇠를 찾으면 열지 못했던 문으로 다시가서 열수 있는지 확인한다.
5. 문서를 찾으면 카운트를 증가시키고, 빈공간으로 변경한다.
