### 풀이
1. 일반적인 bfs만으로는 풀수 없었다.
2. 벽을 부술경우 더 빠른 경우가 있기 때문이다.
3. 한시간 반 정도 고민을 하다가 isVisited를 3차원으로 풀었다는 힌트를 보고 풀었따.
4. isVisited를 3차원으로 만들어줬고, isVisited중 한칸은 벽이 부숴진적이 있는지 없는지를 체크했다.
