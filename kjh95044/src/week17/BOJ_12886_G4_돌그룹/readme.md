## 풀이 🎈


1. c는 sum에서 a,b를 빼서 구한다. (반복을 줄이기 위함)
2. 2차원 visited 배열을 할용해 방문하지 않았다면 문제 조건에 따라 단계를 실행해준다.
3. a,b 와 b,c 그리고 a,c 까지 모두 진행해주고, a,b,c가 같은 경우가 생기면 1을 출력하고 끝낸다.
4. visited 할수 있는만큼 다 방문했는데도, 못찾으면 0을 출력하고 끝낸다.
