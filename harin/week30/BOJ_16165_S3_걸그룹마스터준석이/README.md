### 풀이 과정 👀 
- HashMap 자료구조를 사용해서 문제를 푼다.
- HashMap을 <String, String[]> 구조로 선언해서 String에는 걸그룹 이름, String[]에는 걸그룸 멤버 이름 배열을 넣는다.
- type이 0이거나 혹은 1일때 if문을 써서 원하는 값을 찾아 StringBuilder에 append해준다.
