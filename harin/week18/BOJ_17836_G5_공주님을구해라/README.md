### 풀이 방법 👀 
- 용사는 상하좌우로 이동, 즉 인접한 곳으로 이동이 가능하다고 해서 BFS로 풀었다.
- 그람(검)을 가진 경우와 가지지 않은 경우를 나눠서 체크하여 풀었다. (방문 배열을 3차원으로 만듦)
- 이동 시간이 제한 시간을 넘어설 경우 검사하지 않고 continue
- 공주님에게 도착했을 경우, 현재의 최단 시간과 이동 시간을 비교한 후 작은 값을 ans(답) 변수에 넣어준다.
