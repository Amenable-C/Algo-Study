### 👩‍🏫 풀이
1. Hashmap을 이용해 그룹의 이름과 멤버 목록을 저장한다.
    1-1. 멤버 목록을 사전 순으로 출력해야 하기 때문에 미리 정렬하고 넣어둔다.

    2-1. 0일 경우 멤버들을 출력하면 되기 때문에 key값으로 가져온 멤버목록을 출력한다.
    2-2. 1일 경우 key 목록들을 가져와서 하나씩 멤버 이름이 포함되었는지 확인한다.
