# 5052 G4 전화번호 목록
##### <p align="right"> 2023.05.12 📆 </p> 


### 👩‍🏫 풀이
1. N개의 전화번호를 입력받아 list에 넣는다.
2. list를 정렬한다. (문자열)
3. 비교 문자열보다 길이가 긴 경우에만 앞의 문자열만큼에서 같은지 판단한다.
3-1. 비교 문자열과 동일할 경우 일관성을 유지하지 못해 끝낸다.
4. 비교 문자열을 현재 문자열로 바꾼다.
5. 위의 과정을 반복하고 출력한다.
