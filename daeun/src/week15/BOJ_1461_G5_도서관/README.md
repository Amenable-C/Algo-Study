### 👩‍🏫 풀이
1. 책의 위치를 입력받으면서 양수인 경우와 음수인 경우를 따로 저장한다.
2. 양수와 음수 리스트를 정렬한다.
3. go 를 이용해서 거리를 구한다.
3-1. 양수, 음수 따로 한다.
3-2. 한 번에 갈 수 있는 M번째마다 이동하는 거리의 2배를 더한다. (갔다 왔다)
4. 위치 중 가장 멀리있는 값을 마지막으로 가기 때문에 결과에서 뺀다.
