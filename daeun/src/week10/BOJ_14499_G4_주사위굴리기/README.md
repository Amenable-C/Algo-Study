# 14499 G4 주사위 굴리기
##### <p align="right"> 2023.04.06 📆 </p> 

 
### 👩‍🏫 풀이
1. 주사위 배열은 상단, 하단, 동, 서, 남, 북 순으로 조작했다.
2. 입력받은 명령에 따라 주사위에서 바꾸는 4부분의 값의 바꿨다.
3. 도착한 바닥이 0이면 바닥에 주사위 하단의 숫자를 바닥에 넣는다.
4. 도착한 바닥이 숫자라면 주사위 하단에 해당 숫자를 저장하고 바닥을 0으로 만든다.
5. 바뀐 위치로 반복문을 돈다.
