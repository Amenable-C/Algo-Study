
## 성능 요약
메모리: 11484 KB  
시간 : 88 ms  
소요시간 : 1시간


## 접근 방법
dp[i][j]에 input1의 0~i번째와 input2의 0~j번째까지 중 가장 긴 부분 공통 문자를 저장  
처음 초기값 저장 후 이중 for문을 통해 순회하면서  
input1과 input2의 문자가 같으면 이전 최장문자열에 현재 문자 추가하고  
현재 input1과 input2의 문자가 다르면 이전 최장문자열 중 더 길이가 긴 문자 저장
