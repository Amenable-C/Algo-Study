
## 성능 요약

## 접근 방법
- 문제에서 땅은 아래에서 위로 순서가 매겨지지만 난 위에서 아래로 순서가 매겨지게 함  
	- 로봇의 상하 방향이 반대로 바뀜  
	- 명령어 LR의 화전 방향이 반대로 바뀜  
- map에 로봇의 number를 입력 & robots[]배열의 index에 로봇 정보 저장  
- 명령어들을 실행  
	- 하나의 명령어 실행 끝난 후 기존의 로봇 정보 초기화 및 업데이트(map과 robots수정)  
	- 명령어 실행도중 다른 로봇을 만나거나 범위를 벗어나면 break
