## 접근 방법
1부터 g까지 도킹 가능하므로 최대값인 g에 도킹하고 다음에 g가 들어오면 g-1에 도킹하게 함  
g가 2번째 들어올 때 g-1에 도킹해야하므로 이를 알려줘야함  
서로소집합을 사용하여 다음에 g가 떠 들어오면 g의 부모노드를 g-1로 union  
위의 과정을 반복하며 g의 부모노드가 0이면 더이상 도킹할 수 없으므로 break
