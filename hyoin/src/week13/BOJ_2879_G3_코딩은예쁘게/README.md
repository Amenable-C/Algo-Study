## 접근 방법
같은 그룹이 되는 조건
1. 입력과 결과 차이의 부호가 같아야 함  
	-> 연속된 줄이 (현재 탭의 개수>올바른 탭의 개수)이거나 (현재 탭의 개수<올바른 탭의 개수)여야만 함  
2. 같은 그룹에서도 오름차순이거나 내림차순이어야함  
	- 양수일 때에는 오름차순이어야하고 음수일 때는 내림차순이어야함  
	- 양수일 때 오름차순이면 해당 그룹의 최대값이 결과 & 내림차순이면 이전값과의 차이가 결과
