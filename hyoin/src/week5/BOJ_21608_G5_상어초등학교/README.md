## 접근 방법

자리 선정 기준
1. 좋아하는 학생이 인접한 칸에 가장 많은 칸
2. 인접한 칸 중에서 비어있는 칸이 가장 많은 칸
3. 행의 번호가 가장 작은 칸
4. 열의 번호가 가장 작은 칸
