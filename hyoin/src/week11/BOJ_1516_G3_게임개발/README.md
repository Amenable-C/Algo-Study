## 접근 방법
위상정렬 + dp 사용  
저번에 풀었던 ACMCraft와 똑같은 문제라서 같은 접근 방법이다  
먼저 adjList에 순서에 맞춰 그래프를 작성하고 위상정렬을 진행  
위상정렬 과정 중 순서가 정해질 때 기존에 해당건물을 건설하기 위해 기다려야하는 시간(dp[temp.vertex])보다 현재 cur에서 걸리는 시간(dp[cur] + D[temp.vertex])이 크다면 업데이트 했다
